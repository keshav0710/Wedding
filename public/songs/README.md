# 🎵 Wedding Songs Folder

Drop your MP3 files into this folder with the EXACT filenames below.
The app will automatically pick them up — no code changes needed!

## Required File Names

| Ceremony | Filename         | Suggested Song              |
|----------|------------------|-----------------------------|
| Haldi    | `haldi.mp3`      | Haldi Wale Haath — Neha Kakkar |
| Mehendi  | `mehendi.mp3`    | Mehendi Laga Ke Rakhna      |/
| Sangeet  | `sangeet.mp3`    | Aaj Ki Raat — A.R. Rahman   |
| Mayara   | `mayara.mp3`     | Shubh Vivah / Folk Bhajan   |
| Reception| `reception.mp3`  | Tujh Mein Rab Dikhta Hai    |

## Notes
- Supported format: **MP3** (recommended), also OGG / WAV  
- Keep clips short (1–3 minutes) for best mobile performance  
- Files placed here are served at `/songs/<filename>.mp3`  
- The app shows a friendly "⚠ Song not added yet" message until you add the files  

## How to trim a song
Use **Audacity** (free) or https://mp3cut.net to cut just the lines / chorus you want!
